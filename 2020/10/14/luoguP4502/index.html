Delaunay三角剖分

对于任意一个平面上的离散点集，三角剖分定义为一种近乎完美的剖分方式满足：

  1. 空圆性：任意四点不能共圆。任意一个三角形的外接圆内不得含有除去三角形顶点的其他离散点。
  2. 最小化最大角：在点集的所有三角剖分中三角形最小角最大的剖分。

将一个泰森多边形每个顶点属于的区域的离散点连成三角形，就可以得到离散点的唯一三角剖分了。

  • 前提建立在泰森多变形的顶点不同时属于四个区域，即没有任何四个离散点共圆。

正题

考虑将题目变得人话：

给定一个半径为 的反演圆，圆心在一个矩形内出现概率相等，求点集内所有点对于反演圆反演后的点构成的凸包的顶点数的期望。

考虑到每个点反演后凸包的点数并不是很好求（因为还要带上期望），所以我们考虑通过三角剖分的相关知识，得到凸包点数与三角剖分的转化。

对于Delaunay三角剖分的空圆性质，我们可以将三角形的个数转化为空圆的个数。

接下来考虑空圆反演后对答案的贡献的影响。

我们定义支配圆为圆周上或者圆内包含点集中的所有点的圆，可以发现，支配圆与空圆概念几乎完全相反。

对于一个空圆或者支配圆的反演，我们分两类讨论：

  1. 对于一个反演中心在空圆外的空圆，它的反演图形为一个空圆，否则是一个支配圆；
  2. 对于一个反演中心在支配圆外的支配圆，它的反演图形为一个空圆，否则是一个支配圆。

inline db angle(point x,point y){return atan2(x^y,x&y);}
inline db calc(point x,point y,db r){
    point p=(x-y)/(x-y).norm();
    point q=p; p.inv();
    point mid=intersection(x,y,point{0,0},p);
    if(mid.norm()>r) return r*r*angle(x,y)/2;
    db d=sqrt(r*r-mid.norm2());
    point w1=mid+(q*d),w2=mid-(q*d);
    int b1=eq(x.norm2(),r*r)==1,b2=eq(y.norm2(),r*r)==1;
    if(b1&&b2){
        if(eq((x-w1)&(y-w1))<=0)
            return r*r*(angle(x,w1)+angle(w2,y))/2+(w1^w2)/2;
        return r*r*angle(x,y)/2;
    }
    if(b1) return (r*r*angle(x,w1)+(w1^y))/2;
    if(b2) return (r*r*angle(w2,y)+(x^w2))/2;
    return (x^y)/2;
}

void convex_hull(){
    f[++cnt]=plane(1,2,3),f[++cnt]=plane(3,2,1);
    rep(i,4,n){
        int num=0;
        rep(j,1,cnt){
            int ty=isplane(f[j],poi[i]);
            if(!ty) cpy[++num]=f[j];
            rep(k,0,2) vis[f[j].id[k]][f[j].id[(k+1)%3]]=ty;
        }
        rep(j,1,cnt){
            rep(k,0,2){
                int x=f[j].id[k],y=f[j].id[(k+1)%3];
                if(vis[x][y]&&!vis[y][x]) cpy[++num]=plane(x,y,i);
            }
        }
        rep(j,1,num) f[j]=cpy[j]; cnt=num,num=0;
    }
}
